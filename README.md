# 🏔️ Mountain Explore

A beautiful, responsive **trekking & adventure website** built using plain HTML, CSS and JavaScript.  
This project showcases multiple adventure pages like Home, About, Trips, Gallery, and Contact — complete with an interactive UI and smooth navigation.

Live Site:  
🔗 https://hena82.github.io/Mountain-Explore/

---

## 📌 About The Project

Mountain Explore is a **frontend web project** that presents an exploration-focused adventure site. It includes information about trips, a gallery of adventure images, and a contact form — all styled using modern CSS and enhanced with JavaScript for interactivity.

Perfect for showcasing **responsive design** and coding fundamentals.

---

## 🚀 Features

✔️ Responsive navigation for mobile and desktop  
✔️ Multiple content pages (Home, About, Trips, Gallery, Contact)  
✔️ Trip sections with interactive details  
✔️ Image gallery  
✔️ Smooth scrolling and JavaScript enhancements  
✔️ Clean UI design using CSS and fonts

---

## 🧱 Technologies Used

The project was built using:

- **HTML** – Page structure  
- **CSS** – Styling and layout  
- **JavaScript** – DOM interactions  
- **Google Fonts + Font Awesome** – Typography and icons
